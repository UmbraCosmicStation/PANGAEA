# 🎨 PANGAEA Design System Guide v1.0

## 1. 비주얼 테마: "Deep Ocean & Civilization"
* **Primary Color:** Deep Ocean Blue (`#0F2027` ~ `#203A43`)
* **Accent Color:** Aurora Green (`#00F260`), Magma Red (`#FF416C`), Coral Neon (`#0575E6`)
* **Metaphor:** 모든 데이터는 대륙(Active)이거나 심해의 유적(Archive)이다.

## 2. 아틀란티스 프로토콜 (Aging Visuals)
타일의 상태에 따른 비주얼 변화 가이드.

| 단계 | 상태 | 비주얼 키워드 | 색상 톤 | 오브젝트 |
| :--- | :--- | :--- | :--- | :--- |
| **Lv 1** | **Active** (활성) | 문명, 생명 | Green / Brown | 숲, 마을, 화산, 구름 |
| **Lv 2** | **Warning** (7일~) | 침수, 습기 | Desaturated | 해변 잠김, 젖은 땅 |
| **Lv 3** | **Legendary** (30일~) | 심해, 유적 | Deep Blue / Neon | **형광 산호초**, 물고기, 기포 |

## 3. 나스카 코드 디자인 (Nasca Code)
* **컨셉:** **"성좌 (Constellation)"** - 하늘의 별자리를 땅에 새기다.
* **스타일:**
    * **Line Art:** 끊어지지 않는 벡터 라인 (Circuit style).
    * **Nodes:** 데이터 거점(별)을 의미하는 원/다각형 포인트.
    * **Geometry:** QR의 노이즈 패턴 배제, 기하학적 미로 형태.
* **적용 예시:**
    * 티셔츠 가슴 중앙 금박 라인 아트.
    * 포토카드 뒷면 홀로그램 박.
