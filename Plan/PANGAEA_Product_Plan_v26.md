# PANGAEA (판게아) 제품 기획안 — v26 (Final)

**캐치프레이즈:** “당신의 대륙을 탐험하세요 (Discover Your Continent).”
**제품명:** PANGAEA (판게아)
**버전:** v26
**작성일:** 2025-12-07

## 0. 핵심 전략 (Core Strategy)
* **타겟:** 크리에이터(스트리머) & 팬덤 (Primary) / 개발자 (Secondary).
* **가치:** 커뮤니티 화력의 시각화(영토), 데이터 주권(GitHub), O2O 연결(나스카).
* **철학:** "데이터는 지층처럼 퇴적되어 나를 지탱하는 대륙이 된다."

## 1. 홈 화면: 대륙과 문명
### 1.1 듀얼 뷰 엔진 (Dual View Engine)
* **A. 위성 뷰 (2D):** 직관적 정보 전달을 위한 평면 히트맵 (면적=총량, 색=활성도).
* **B. 지형 뷰 (3D Isometric):** 탐험과 몰입을 위한 **'아틀란티스'** 스타일의 등고선 지형.
    * **고도:** 데이터 총량에 비례하여 산맥 형성.
    * **상태:** 활성 시 화산 활동/연기 이펙트.

### 1.2 시각적 노화 시스템 (Atlantis Protocol)
데이터의 수명 주기에 따라 타일의 비주얼이 자동 진화.
1.  **활성기:** 푸른 숲, 동물, 화산 (Live).
2.  **침수기:** 해수면 상승, 해변 침수 (Warning).
3.  **심해기 (퇴적):** 완전 침수 후 **'형광 산호초 유적'**으로 변모 (Legendary).

## 2. 정보 구조 & 기술 아키텍처
### 2.1 하이브리드 저장소 (Hybrid Storage)
* **Hot Data (채팅):** Redis(In-memory) + MongoDB. 초고속 실시간 처리.
* **Cold Data (문서):** GitHub Repository. 영구 보존 및 버전 관리.
* **동기화:** 매일 자정, 채팅 로그를 `.md`로 변환하여 GitHub에 아카이빙(퇴적).

### 2.2 나스카 코드 (Nasca Code)
* **기술:** 폐쇄형 AR 비주얼 마커 (보안성 강화).
* **디자인:** **성좌형(Constellation)**. 노드와 링크로 구성된 기하학적 회로도.
* **UX:** 앱 내 카메라로 스캔 시 3D 웜홀 이펙트와 함께 특정 좌표(방)로 이동.

## 3. 경제 모델 (TER Economy)
**"퇴적과 영토 (Sediment & Territory)"**
* **삭제 = 영토 축소:** 무료로 삭제 가능하나, 대륙 면적(랭킹)이 즉시 줄어듦.
* **퇴적 = 영토 보존:** 오래된 데이터는 심해로 가라앉지만(아틀란티스화), 영토 면적은 유지됨.
* **BM (토큰 사용처):**
    * **인양(Salvage):** 심해의 데이터를 수면 위로 복구.
    * **박제(Pinning):** 영원히 빛나는 모노리스로 고정.
    * **꾸미기:** 산호초 스킨, 나스카 랜드마크 건설.

## 4. 로드맵 (Roadmap)
* **Phase 1 (MVP):** PixiJS 기반 2D/3D 뷰 엔진, GitHub 연동, 퇴적 시스템.
* **Phase 2 (Alpha):** 나스카 AR 스캐너 구현, 굿즈 샘플링, O2O 마케팅.
* **Phase 3 (Beta):** 스트리머 오버레이 SDK, 토큰 경제 도입.
